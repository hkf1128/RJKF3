### 2. Anaconda（Jupyter Notebook）安装

1. 下载 Anaconda 安装包

    https://www.anaconda.com/download/  下载 或者使用镜像源下载

2. 安装配置：

    - 路径无中文、无空格
    - 选择 just for me 否则需要管理员权限
    - 使用 Anaconda 作为默认 Python 环境

 <img width="1096" height="896" alt="image" src="https://github.com/user-attachments/assets/143bf620-1212-49aa-8ec4-b8c8d4c43ae0" />


    配置环境变量 

    将安装的地址的加入到系统的环境变量

    ```
    D:\anaconda3
    D:\anaconda3\Scripts
    D:\anaconda3\Library\bin
    D:\anaconda3\Library\mingw-w64\bin
    ```

    

3. 安装验证：

     搜索prompt ,点击anaconda prompt 或者anaconda Powershell Prompt,输入conda  --version

<img width="511" height="132" alt="image" src="https://github.com/user-attachments/assets/b4b7a0ae-c12b-4747-8050-6101f7ee2202" />

    

    

    

4. 安装和启动 Jupyter Notebook：

    在anaconda 的环境下安装Jupyter Notebook

    4.1 创建虚拟环境 

    ```
    conda create --name jupyter_learn 替换为自己的取名的名字
    conda env list 查看是否创建成功
    ```

<img width="545" height="236" alt="image" src="https://github.com/user-attachments/assets/a464ef76-76b4-4f94-881e-bb3f7071fdd6" />

    看到my_jupter意味着创建成功

    

    4.2 启动

    ```
    jupyter notebook
    ```

<img width="1202" height="711" alt="image" src="https://github.com/user-attachments/assets/3640a630-6577-4f93-97db-f65c63821010" />

    自动跳转到浏览器，安装成功

    

    默认是c盘目录的内容，如何修改呢？

    ```
    jupyter notebook --generate-config
    ```

    找到对于的配置文件.py  修改ServerApp.notebook_dir

    ```
    c.ServerApp.notebook_dir = 'D:/jupyterProjects'
    ```

<img width="1234" height="454" alt="image" src="https://github.com/user-attachments/assets/ec49d134-861d-4e4e-a597-533164ad7d3a" />

  这样，进入的就是定向的文件夹
